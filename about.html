shtfy is a URL-shortifier run by aykit.

We at aykit hate data retention and love open-source. shtfy is keeping storing data to a minimum:

All URLs shortified
For shtfy to work correctly, we need to save every URL that has been shortified and it's shortified version.
Hits
We save how often a shortified URL has been visited. Note that we only save the amount of hits, nothing else.
Piwik visit tracking - Anonymised visits
Knowing how many people visit our service is always good for morale. However, we do our best to track as little as possible: We save IPs hitting the main page (ay.vc) and this page (about.html) with the last digits anonymised (192.168.0.xxx). Additionally, we respect your Browser's Do-Not-Track setting. If you still don't trust us, we recommend usage of Ghostery or similar software.

And that is all. If you want to host your own version of shtfy, go to shtfy on github.
